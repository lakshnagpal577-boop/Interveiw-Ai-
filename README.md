# Interview AI 🚀

**Interview AI** is an AI-powered web application that helps job seekers evaluate their resumes against a target job description. By leveraging Large Language Models (LLMs), the application identifies missing skills, analyzes resume-job compatibility, and generates a personalized roadmap to help users prepare for their desired roles.

## ✨ Features

* 📄 Upload and analyze resumes.
* 💼 Compare resumes with job descriptions using AI-powered semantic analysis.
* 📊 Generate a resume-job match score with detailed feedback.
* 🎯 Identify missing technical and soft skills required for the target role.
* 🗺️ Create personalized learning roadmaps and study schedules based on skill gaps.
* 📈 Interactive dashboard displaying resume insights and improvement suggestions.
* 🔐 Secure user authentication and personalized user profiles.

## 🎯 Objective

Interview AI aims to simplify interview preparation by providing intelligent resume analysis and personalized guidance. Instead of manually comparing resumes with job descriptions, users receive actionable insights, skill-gap analysis, and a structured preparation plan to improve their chances of landing interviews.
